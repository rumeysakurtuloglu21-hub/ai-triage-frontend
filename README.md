# 🏥 AI Triage System

Yapay zeka destekli sağlık ön değerlendirme sistemi.

## 🚀 Proje Hakkında

AI Triage kullanıcıların semptomlarını analiz ederek:

- Risk seviyesini belirler
- Uygun sağlık bölümünü önerir
- Doktora yönlendirme sağlar
- Hasta sağlık profilini yönetir


## 🛠 Kullanılan Teknolojiler

Frontend:

- React
- Vite
- React Router
- JavaScript
- CSS


Backend:

- FastAPI
- REST API
- AI Medical Engine


Database:

- MySQL

## ⚙️ Kurulum

Projeyi indir:

```bash
git clone repo-url

Klasöre gir:

cd ai-triage-frontend

Paketleri yükle:

npm install

Çalıştır:

npm run dev
🔌 API Bağlantısı

Frontend backend API üzerinden haberleşir.

Örnek:

POST /triage
POST /transcribe

Backend adresi .env üzerinden ayarlanacaktır.
